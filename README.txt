To manually clone the project to other devices, follow these steps:

1. Ensure that Python's latest version is installed on your device.
2. Extract the folder named "codes" from the project.
3. Open the command prompt and navigate to the "codes" directory.
4. Run the command `pip install -r requirements.txt` to install all the necessary libraries and frameworks required for the project.
5. After installation, type `streamlit run a.py` in the command prompt.
6. If prompted, enter any random email address.
7. The project will then be executed and displayed in your default web browser.
